# 加入我们

# 如何加入？

Step 1: 关注社群，关注 github 账户， star 对应自己感兴趣 repo.

Step 2: fork 项目，提交 2 次 commit

Step 3: 联系 info@airdb.com, 提名

Step 4: 提名后，团队内部宣告

# 要求

  1. 具备网络环境，熟练使用电脑。
  2. 具备软件开发能力，能够使用 Linux, git， vscode 等工具。
